# Baltic
 
Graphical sea simulator.

## Controls

- `W` - Move forward
- `S` - Move backward
- `A` - Move left
- `D` - Move right
- `Shift` - Move down
- `Space` - Move up
- `Mouse` - Look around
- `Escape` - Pause/Resume
- `F11` - Toggle fullscreen

## Build

### Prerequisites

Ensure you have the following installed on your system:

- CMake 3.28 or later
- DirectX SDK (June 2010) or later
- C++20 compiler (e.g. Clang 19.1.0 or later)
- DXC (DirectX Shader Compiler) 1.8 or later

### Build Instructions

To build the project, run the following commands in the project root directory:

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
